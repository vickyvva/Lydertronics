# Contributing to Lydertronics

We appreciate your interest in contributing to Lydertronics! This guide will help you get started.

## How to Contribute

1. **Fork the repository** and create a new branch for your feature
2. **Make your changes** following our coding standards
3. **Test thoroughly** before submitting
4. **Commit with clear messages** describing your changes
5. **Submit a Pull Request** with a detailed description

## Code Standards

- **HTML**: Use semantic HTML5 elements
- **CSS**: Follow BEM naming conventions, maintain organized structure
- **JavaScript**: Write clean, commented code with consistent formatting

## Reporting Issues

Found a bug? Please create an issue with:
- Clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Browser/environment details

## Pull Request Process

1. Update documentation as needed
2. Add comments for complex logic
3. Ensure all tests pass
4. Request review from maintainers
